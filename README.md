# 🍆 Pisun Bot

**Pisun Bot** — це розважальний Telegram-бот на базі `aiogram 3`, створений для вимірювання віртуальних пісюнів у чатах. Весела гра для друзів з рейтингами, випадковими подіями та цікавими фактами.

## 🚀 Основні можливості

- **🍆 Вимірювання:** Команда `/pisun` дозволяє раз на добу дізнатися свій поточний розмір. Система "справедливого рандому" спочатку допомагає новачкам рости швидше.
- **🏆 Рейтинги:**
  - `/top` — Загальний залік чату.
  - `/top_week` — Хто виріс найбільше за поточний тиждень.
- **🎓 Цікаві факти:** Якщо ви вже мірялися сьогодні, бот поділиться випадковим науковим (або не дуже) фактом про анатомію людей та тварин.
- **🚨 Екстрені події:** Раз на тиждень у випадковий час у чаті може з'явитися "нічийна піхва". Той, хто встигне першим натиснути кнопку, отримає бонус до розміру (або потрапить у пастку!).
- **🛡️ Захист:** Вбудований флуд-контроль (Throttling) для запобігання спаму.

## 🛠️ Вимоги

- **Python 3.10+**
- **SQLite**
- **Docker & Docker Compose** (рекомендовано)

## 📦 Встановлення та запуск

### 1. Налаштування середовища
Створіть файл `.env` на основі прикладу:
```bash
cp .env.example .env
```
Відредагуйте `.env`, додавши свій `BOT_TOKEN` (від [@BotFather](https://t.me/BotFather)) та `ADMIN_IDS`.

### 2. Запуск через Docker (рекомендовано)
```bash
docker-compose up -d --build
```

### 3. Локальний запуск
Встановіть залежності:
```bash
pip install -r requirements.txt
```
Запустіть бота:
```bash
python -m src.main
```

## ⚙️ Змінні оточення (.env)

| Змінна | Опис |
| :--- | :--- |
| `BOT_TOKEN` | Токен вашого бота від BotFather |
| `ADMIN_IDS` | ID адміністраторів через кому (для секретних команд) |
